# Simple ToDo List : Gestionnaire de Tâches Minimaliste

## Introduction

Bienvenue sur le dépôt **Simple ToDo List**, une application web de gestion de tâches rapide et minimaliste. Ce projet a été développé en utilisant **React JS** pour l'interface utilisateur et **Vite** pour un environnement de développement ultra-rapide.

L'objectif était de créer une solution simple et efficace pour l'organisation quotidienne, tout en mettant en pratique les principes fondamentaux de React (Hooks et gestion d'état).


## Fonctionnalités Principales

* **Ajout Instantané :** Créez de nouvelles tâches avec une simplicité maximale.
* **Marquage d'Achèvement :** Cochez une tâche pour la marquer comme complétée.
* **Suppression Facile :** Éliminez les tâches terminées ou non désirées.
* **Interface Utilisateur (UI) Réactive :** L'application est rapide grâce à l'efficacité de React et Vite.

##  Démarrage du Projet (Local)

Pour lancer une copie locale de ce projet pour le développement ou l'expérimentation, suivez les instructions ci-dessous.

### Prérequis

Assurez-vous que [Node.js](https://nodejs.org/) (version 16 ou supérieure recommandée) et `npm` sont installés sur votre machine.

### Installation

1.  **Clonez le dépôt** (remplacez l'URL par la vôtre) :
    ```bash
    git clone https://github.com/Brahimi-Talla01/Todo-List-React-JS.git
    cd Todo-List-React-JS.git
    ```

2.  **Installez les dépendances :**
    ```bash
    npm install
    ...
    ```

3.  **Lancez le serveur de développement :**
    ```bash
    npm run dev
    ```
    L'application devrait être accessible à `http://localhost:5173/` dans votre navigateur. Le rechargement à chaud (HMR) est pris en charge par Vite.

### Build pour la Production

Pour créer une version optimisée et prête à être déployée :
```bash
npm run build
```
